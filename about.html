About Me

Developer. Problem solver. Perpetual learner.

I'm a full-stack developer with a focus on performance and reliability. My primary languages are Go and Rust for backend systems — APIs, CLI tools, and concurrent network applications. For scripting, automation, and data work I reach for Python.

On the frontend I use JavaScript, React, and Tailwind CSS to build clean, responsive interfaces. I care about the details: load times, accessibility, and layouts that work on every screen.

I'm comfortable with Docker, Git, and Linux environments, able to take a project from a local dev setup all the way to deployment. I'm always building something, always learning, and always open to interesting problems.
