About Me

I’m a dedicated full-stack developer equipped with the skills to build efficient and scalable applications. My technical expertise covers both front-end and back-end development, making me versatile in tackling a range of development tasks.

On the back end, I work primarily with Golang, leveraging its speed and efficiency to build reliable APIs and services. For the front end, I use JavaScript, HTML, and CSS, along with frameworks like React and Tailwind CSS to create responsive, visually engaging interfaces that work seamlessly across devices.

I'm also experienced in Docker, Git and Linux/Unix environments, which allows me to manage and deploy applications efficiently. I’m always committed to improving my skills and staying current with industry trends to deliver high-quality, maintainable code.
